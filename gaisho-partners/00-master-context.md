# GAISHO PARTNERS 構想 — マスターコンテキスト

> このドキュメントは全プロンプトの共通前提。営業資料・仕様書・Webサービスの3プロンプトの先頭にそのまま貼り付けて使う。

## 事業背景

株式会社MUFASA Technologyは、富裕層向け外商コンシェルジュ「GAISHO AI」を運営。LINE上でAIが顧客対応し、ラグジュアリー商品の調達・認証・配送までを一気通貫で提供。バックエンドにHIDE TOKYO（在庫・真贋認証・カタログレイヤー）を持つ。

## GAISHO PARTNERS の位置づけ

GAISHO AIの基幹インフラを、外部の「富裕層接点保有者」に開放するプラットフォーム事業。TERASSが不動産仲介で実現した「個人エージェントが明日から開業できる」モデルのラグジュアリー版を目指す。

## GAISHO AI が提供できるインフラ資産

1. **AI調達エンジン** — GAISHO Hunter：多ソース監視・価格比較
2. **海外バイヤーネットワーク** — 並行輸入仕入れチャネル
3. **真贋認証ノウハウ・体制**
4. **HIDE TOKYO 在庫プール** — Shopify / Rakuten 連携済
5. **LINE CRM + Lステップ自動化** — 顧客対応の半自動化
6. **配送・通関・アフターサービス体制**
7. **GAISHOブランドの信頼性・実績**
8. **Claude APIによる接客・商品提案AI**

## ターゲットエージェント像（GAISHO PARTNER）

富裕層に直接アクセスできる個人。

- 元ブランドSA、銀座／六本木のラウンジ嬢・ママ、私設秘書、コンシェルジュ
- 高級ホテル・会員制サロンのスタッフ
- 富裕層向けインフルエンサー、スタイリスト
- ジュエラー、バーテンダー、ゴルフ場スタッフ

**共通項：仕入れも認証も配送もできないが、信頼関係のある顧客リストを持つ。**

## ビジネスモデル仮説

- **パートナー報酬**：成約手数料の30〜50%（紹介ボーナス・継続ボーナスあり）
- **パートナー側参入障壁**：富裕層コネクションのみ（資格不要）
- **月額SaaS**：¥50K〜¥200K/社（B2B：百貨店外商部・コンシェルジュ会社向け）
- **個人パートナー**：固定費なし、成約コミッションのみ

## 競合・差別化

- **直接競合**：なし（日本のラグジュアリーtoAインフラは空白市場）
- **類比**：BUYMAは「個人バイヤー」、GAISHOは「個人外商」
- **差別化軸**：仕入れ・認証・配送の「裏側」を完全代行する点

## トーン＆スタイル

- **ターゲット読者**：富裕層接点保有者（必ずしも事業経験者ではない）
- **言葉**：専門用語を避ける／「稼げる」より「品格ある収入」
- **ビジュアル**：黒×ゴールド／高級感／余白多め／TERASSより圧倒的にラグジュアリー寄り

## 守るべき制約

- 並行輸入であることは隠さない（信頼性が下がる）
- パートナーは「GAISHO代理人」ではなく「GAISHO公認パートナー」と表記
- 真贋保証は GAISHO 側が負う（パートナーに責任転嫁しない）

## 技術スタック（既存）

- フロントエンド：Next.js App Router / TypeScript strict / Tailwind / shadcn/ui
- バックエンド：Cloudflare Workers + Hono + D1
- 認証：Clerk または NextAuth
- 顧客接続：LINE Harness OSS
- 在庫：HIDE TOKYO Shopify Storefront API
- 仕入れ監視：GAISHO Hunter
- リード育成：Lステップ
- AI：Claude API（claude-opus-4-7, claude-sonnet-4-6）
